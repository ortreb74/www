La racine

La racine est le répertoire ela /usr/local/ela.

Les répertoires de premier niveau

ela
bin
trt
cd-rom
trace

Le répertoire ela (/usr/local/ela) contient 27 sous répertoires, dont 4 discutés ici :

Le répertoire bin contient les programmes. Principalement les programmes en shell.

Le répertoire trt est hybride, il contient des programmes par produit. On y retrouve les programmes balises (source et compilés).

Le répertoire cd-rom contient les données.

Le répertoire trace contient des fichiers journaux produits par les traitements.

Le répertoire cdrom est le répertoire source des données. Sur le schéma ci dessous j'ai représenté 6 sous répertoires de cd-rom. Mais cale mérite explication. Il y a de fait bien plus de sous répertoire dans cd-rom que ceux représentés : dp00 dp15 sont une selection de produit parmi une liste bien plus large ; de même gp20 à gbp75.

On a ainsi 3 niveaux de données

  • les données après leur redaction datacomm est le répertoire des textes, dpxx (ou gpxx) sont les données (produit des EL)
  • un autre répertoire à l'intérieur des produits qui correspond au format attendu par le premier front
  • hulk est le répertoire des données au format attendu côté front

La chaîne de traitement

Il s'agit d'un programme écrit en shell (tcsh) qui copie les données depuis la source (datacomm) vers la cible (un répertoire qui porte le nom du produit)

Ce programme n'existe que sous sa forme principale : la production

Sa structure est un gros bloc conditionnel (case ou choose).

Le programme fait 907 lignes.
